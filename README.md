### 1. `df.info()`
- Returns a concise summary of the DataFrame.
- Shows:
  - Column names
  - Non-null counts
  - Data types
  - Memory usage

```python
df.info()
```

---

### 2. `df.describe()`
- Returns summary statistics of numeric columns.
- Includes:
  - Count
  - Mean
  - Std
  - Min, Max
  - 25%, 50%, 75% Quartiles

```python
df.describe()
```
